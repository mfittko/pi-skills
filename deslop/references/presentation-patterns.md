# Hard Deslop Reference Patterns

Use these replacements as defaults, not rigid templates.

## Contrast phrasing

Avoid:
- It is not X, it is Y.
- This is not X. This is Y.
- The point is not X.
- Not X but Y.

Prefer:
- State Y directly.
- Name the actual mechanism.
- Name the actual payoff.

Examples:
- Instead of `This is not orchestration. It is a shipping process.`
- Use `This shipping process moves work through the loops required to ship.`

## Empty headlines

Avoid:
- Executive summary
- Overview
- What this is
- Key takeaways
- Final thoughts

Prefer:
- Why this matters
- Where time is lost
- The loop architecture
- Tracker-first model
- Company impact

## Generic business fog

Avoid:
- enables
- empowers
- facilitates
- drives alignment
- operationalizes
- unlocks value

Prefer:
- moves
- resumes
- closes
- checks
- routes
- updates
- ships

## Deck tightening moves

When a slide feels bloated:
1. shorten the title
2. cut one-third of bullets
3. shorten labels before shrinking fonts
4. turn paired lists into two columns
5. split one overloaded diagram into two simpler diagrams

## Good test

After rewriting, ask:
- Would a sharp human say this out loud?
- Is the slide making a claim, or announcing a topic?
- Can I delete this sentence without losing meaning?
- Am I explaining the process, or naming the actual step?
