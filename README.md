# skills

Reusable Pi agent skills.

## presentation-builder

Create polished Slidev pitch decks using a parallel-variant chain workflow. Generates a contextual briefing, fans out 3 variants per slide, picks/combines the best, runs story-coherence and deslop review passes, and validates layout with Playwright screenshots.

See [presentation-builder/SKILL.md](presentation-builder/SKILL.md) for the full workflow.
